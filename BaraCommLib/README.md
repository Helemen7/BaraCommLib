# BaraCommLib: A commercial-like Python library for Capybara robot

## Why is this needed?
Even tho the entire robot codebase is written in High Performance C++, I decided to create this library because of the Technical Challenge, where you need to develop an almost entirely different code for the robot. Because my C++ code isn't so easy I went over to python. 

## What should this library accomplish?
This lib should be as easy as a Makeblock, EV3 etc to program, with also some CV features, that will make this incredibly difficult.

_Note from the author: No pain, no gain._
